# Task Management API

A RESTful API for task management built with Node.js, Express, TypeScript, and Prisma.

## Features

- User authentication and authorization
- Task management (CRUD operations)
- Task categorization
- Task comments
- Role-based access control

## Tech Stack

- Node.js & Express
- TypeScript
- Prisma ORM
- PostgreSQL
- JWT Authentication
- Jest for testing

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- PostgreSQL
